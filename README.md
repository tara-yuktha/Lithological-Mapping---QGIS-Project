# Lithological Mapping of the Malwa Plateau — QGIS Project

## Overview

This project presents a basin-scale lithological map of the **Malwa Plateau**, central India, covering parts of Madhya Pradesh and Rajasthan (approximately 74°E–78°E, 22°N–25°N). The map was produced using QGIS as part of MS thesis research on **CO₂ mineralisation and sequestration potential in Deccan Trap basaltic formations** at IISER Bhopal.

The Malwa Plateau is underlain predominantly by Deccan Trap basalts — one of the largest flood basalt provinces in the world — making it a geologically significant region for carbon capture and storage (CCS) research. This map forms the spatial foundation for field sampling campaigns, outcrop characterisation, and reactive transport modelling of CO₂–basalt interactions.

## Project Highlights

- Mapped **6 lithological units** across the Malwa Plateau: Basalt, Alluvium, Sandstone, Shale/Mudstone, Granite, and Other
- Integrated **geological polygon data** with satellite basemap imagery for spatial context
- Marked **field sampling locations** (Locations 4, 5, 6) in the Mhow–Pithampur area — sites selected for XRD, petrographic, and petrophysical characterisation
- Included **road network overlay** (NH47, SH28) for logistical field planning
- Produced a **publication-quality cartographic layout** with legend, scale bar, north arrow, and coordinate grid
